## 🔧 Installation

For the MCP server:
```bash
npm install -g b00t-mcp
```

For the full CLI experience, clone the repository:
```bash
git clone https://github.com/elasticdotventures/dotfiles.git
cd dotfiles
# Follow setup instructions in the repo
```
